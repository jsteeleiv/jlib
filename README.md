# jlib
jlib is my personal, primitive standard library.

It is a growing collection of small, focused utilities written in the languages I use over time. The goal is not to compete with large frameworks or production-grade libraries, but to build a foundational toolkit that reflects how I think and work.

# structure
Each language lives in it's own directory.

For C libs, headers follow a per-module implementation guard pattern

# philosophy

> Primitive over clever.
> Explicit over magical.
> Personal over universal.

This is my `__toolbox__`.
