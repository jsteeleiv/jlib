# structure
Each language lives in it's own directory.

## C libraries

Each module has its own header and follows a single-header implementation pattern.
> inspired by [nothings/stb](https://github.com/nothings/stb).

For more info, look into the [C preprocessor](https://en.wikipedia.org/wiki/C_preprocessor)

# philosophy

> Primitive over clever.
> Explicit over magical.
> Personal over universal.

This is my `__toolbox__`.
