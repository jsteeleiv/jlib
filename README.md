# structure
Each language lives in it's own directory, named according to the language.

##### C libraries

Each module has its own header and follows a single-header implementation pattern.
> inspired by [nothings/stb](https://github.com/nothings/stb).

For more info, look into the [C preprocessor](https://en.wikipedia.org/wiki/C_preprocessor)

- One of the goals is to determine _exactly_ where the namespacing will break, to learn more about how I can intertwine with it efficiently and elegantly.
